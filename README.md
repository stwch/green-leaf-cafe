# GreenLeafCafe
このサイトは、店舗情報を学習した ChatGPT に制作依頼をしてもらい、制作したポートフォリオサイトです。<br />
IT に詳しくない設定でヒアリングを行い、要件、キャッチコピー、テキストなどを定義しました。<br />
ブログ記事、イベント内容、画像などは、AI がユーザーとして生成したものを使用しています。

サイト → 現在、制作中です

<br />
<br />

## 機能
- イベント予約・管理システム
- ブログ ← 現在、WordPress で構築中
